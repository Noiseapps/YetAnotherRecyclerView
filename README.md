# Using

1. Add jitpack.io to maven repositories in gradle:


```groovy
allprojects {
        repositories {
            jcenter()
            maven { url "https://jitpack.io" }
        }
```

2. Add to your project dependencies:

```groovy
dependencies {
    compile 'com.github.Noiseapps:YetAnotherRecyclerView:0.0.1'
}
```
